# Word loud

### Installation:

```
pip3 install jieba
```

Download and install Visual CPP Build Tool:
http://landinghub.visualstudio.com/visual-cpp-build-tools

```
pip3 install wordcloud
pip3 install Image
pip3 install numpy
pip3 install matplotlib
```

Run the app.
```
python3 run.py
```
